🎯 COMO USAR A SIMULAÇÃO:
───────────────────────────────────────────────────────────────

1️⃣  ABRA O SCORE CALCULATOR (painel no topo da tabela)
   → Clique para expandir
   → Selecione um curso para ver o cálculo detalhado

2️⃣  AJUSTE OS PESOS DOS CRITÉRIOS (sidebar direito)
   → Mova qualquer slider de 0.0 a 2.0
   → Veja as BARRAS no Score Calculator mudarem!
   → Veja os SCORES na tabela atualizarem!
   → Veja a ORDEM mudar!

3️⃣  AJUSTE OS COUNTRY MODIFIERS (sidebar esquerdo)
   → Cada país tem seu próprio slider (0.5x - 1.5x)
   → Aumente Brazil para 1.5x = cursos BR sobem no ranking
   → Reduza USA para 0.7x = cursos USA caem no ranking
   → MUDANÇAS INSTANTÂNEAS!

4️⃣  USE O BOTÃO "TEST"
   → Aplica pesos extremos alternados
   → Mostra mudanças DRAMÁTICAS
   → Perfeito para demonstrações

5️⃣  TROQUE A FUNÇÃO
   → AE, BA ou DS/MLE
   → Cada uma tem multipliers diferentes
   → Rankings mudam completamente!

───────────────────────────────────────────────────────────────
🎨 INDICADORES VISUAIS:
───────────────────────────────────────────────────────────────

Nos Pesos:
  🟢 Verde com ↑ = peso aumentado (>1.0)
  🟠 Laranja com ↓ = peso reduzido (<1.0)
  ⚪ Cinza = peso padrão (1.0)

Nos Country Modifiers:
  🟢 Verde = boost (>1.0)
  🟠 Laranja = redução (<1.0)
  🟣 Fundo roxo = valor foi modificado

Na Tabela:
  📊 Barras azuis = Base Score
  📊 Barras roxas = Final Score
  🟢/🟠 Modifier colorido
  ✨ Animações suaves (300ms)
